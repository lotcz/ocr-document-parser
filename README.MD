- # OCR Document Parser

## Setup

### Download Tesseract language models

Download from https://github.com/tesseract-ocr/tessdata or https://github.com/tesseract-ocr/tessdata_best.

Place into `home/tessdata`.

## Run

Run `bin/build` and `bin/run`.
